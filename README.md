🇷🇺 Описание

Telegram Cinema Bot — это Telegram-бот для поиска и подбора фильмов по жанрам, настроению и пользовательским предпочтениям. Бот помогает быстро найти интересное кино без долгого поиска и бесконечного выбора.

🇬🇧 Description

Telegram Cinema Bot is a Telegram bot for searching and recommending movies based on genres, mood, and user preferences. The bot helps users quickly find interesting movies without spending hours choosing what to watch
